# Privacy Policy Draft

Effective date: 2026-05-16

`彰師大公告好讀` is a Chrome extension that improves readability for supported National Changhua University of Education electronic bulletin pages.

## Data Collection

This extension does not collect, transmit, sell, or share user data.

The extension reads the content of supported NCUE bulletin pages only inside the user's browser so it can render a clearer reading layout. The developer does not receive this page content.

## Local Storage

The extension uses Chrome's local storage only to remember whether the user has enabled or disabled the reader enhancement.

The extension does not store bulletin content, attachment content, browsing history, credentials, names, emails, or analytics data.

## Network Requests

The extension does not send requests to developer-controlled servers.

When a user clicks a bulletin link or attachment, Chrome may navigate to the target URL in the normal browser flow. Those links are controlled by the original bulletin page, not by this extension.

## Permissions

The extension requests the following permissions:

- `storage`: stores the local enable/disable preference.
- `https://aps.ncue.edu.tw/*`: enables the reader enhancement on supported NCUE APS bulletin pages.
- `https://apss.ncue.edu.tw/*`: enables the reader enhancement on supported NCUE APSS bulletin pages.

The extension's content scripts are limited to NCUE bulletin detail pages under `/odedi/doc_page.php*`.

## Third Parties

This extension does not use analytics, advertising SDKs, tracking pixels, or third-party data processors.

## Changes

This privacy policy may be updated when the extension changes. Any material change should be reflected in the Chrome Web Store listing and in this document.
